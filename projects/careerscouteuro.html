CAREERSCOUTEURO

Strategize your next move. Across borders.

SITUATION
  • International students (especially Indian students in the US) exploring European job markets face fragmented information across job boards, Reddit, and university sites
  • No single tool combined job scraping, currency comparison, and community insights for European relocation planning
TASK
  • Build a dashboard that aggregates European AI/ML/SWE job opportunities with supporting data (currency, universities, community insights)
  • Filter for practical constraints: no French requirement, no recommendation needed
  • Cache results and support background scraping for performance
ACTION
  • Built Flask backend with endpoints for European currency comparison, GitHub repo search, university data, job board scraping, and Reddit community scraping
  • Implemented filtering for language requirements and recommendation policies
  • Added caching layer and background scraping for responsive UX
  • Frontend dashboard with HTML/CSS/JS
RESULT
  • Working prototype that aggregates jobs, currency data, university info, and Reddit insights in one dashboard
  • Practical filters for real constraints international students face
  • Designed for personal use and shared with peers navigating European job markets

TECH STACK

Python Flask JavaScript CSS HTML Web Scraping
