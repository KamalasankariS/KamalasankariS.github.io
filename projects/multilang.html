MULTILANG CODE PARSER DATASET

7 million records. 10 languages. One schema.

SITUATION
  • AI code models are typically trained on raw source code without structural understanding
  • No large-scale, cross-language dataset existed with unified AST representations
  • Researchers needed a common schema to study syntax-aware code intelligence across languages
TASK
  • Parse source code from 10 programming languages into a unified AST representation
  • Scale to millions of records while maintaining 99.9%+ parse success rate
  • Publish openly for the research community
ACTION
  • Used TreeSitter to parse code from 10 languages into universal AST format
  • Processed 7,021,722 records (~114 GB) in Parquet format
  • Built an AST visualization tool for exploring the dataset
  • Co-authored with Jugal Gajjar, published on arXiv (2510.16357) and hosted on HuggingFace
RESULT
  • Published on arXiv, dataset hosted on HuggingFace
  • 7M+ records, 10 languages, 99.9999% successful AST conversions
  • Enables cross-language code representation learning, syntax-aware LLM pretraining, clone detection, and bug prediction
  • Largest unified multi-language parsed code dataset available

TECH STACK

Python TreeSitter HuggingFace Parquet Jupyter LaTeX
