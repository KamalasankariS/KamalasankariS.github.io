LILLUP SCRIBE

20+ formats. One universal schema.

SITUATION
  • The Lillup iOS app needed to ingest content from 20+ file formats (docx, xlsx, pdf, epub, mobi, mp3, Apple iWork, Google Docs references)
  • Each format has different parsing requirements, and no single library handles all of them
  • The output needed to be a standardized JSON schema for downstream processing in the app
TASK
  • Build a modular file parser that converts any supported format into a unified JSON structure
  • Design for both offline (pure Python) and cloud-powered (with web interface) deployment
  • Ensure metadata extraction alongside content parsing
ACTION
  • Designed a dispatcher architecture: dispatcher.py routes files to 13 format-specific parsers
  • Each parser extracts metadata and content into a unified JSON schema
  • Used PyMuPDF, python-docx, openpyxl, python-pptx, BeautifulSoup, mutagen, ebooklib, and more
  • Cloud version adds HTML/JS/CSS web interface for upload and preview
RESULT
  • Supports 20+ file formats with unified JSON output
  • Modular design allows easy addition of new format parsers
  • Deployed as part of the Lillup Scribe iOS app pipeline
  • Both offline and cloud-powered versions available

TECH STACK

Python PyMuPDF python-docx openpyxl BeautifulSoup mutagen ebooklib
