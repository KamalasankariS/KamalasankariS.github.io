SEMANTIC MALWARE DETECTION

Two-phase. High-precision. No mercy.

SITUATION
  • Malicious code can be disguised within seemingly benign programs
  • Traditional static analysis tools miss semantically obfuscated threats
  • No existing tool combined intent comparison with program decomposition for deep inspection
TASK
  • Build a two-phase system: first compare code behavior against stated intent, then decompose suspicious programs into independent components for targeted security inspection
  • Use LLMs for semantic understanding and similarity scoring
  • Provide explanations and fix recommendations, not just binary classification
ACTION
  • Phase 1: Used DeepSeek Coder v2 (1.3B) to generate code descriptions, compared with user-provided intent using GTR-T5-Large for semantic similarity
  • Phase 2: Decomposed programs via AST parsing and graph analysis into independent components
  • Scored each component (1-10 robustness); components below threshold underwent deep security inspection for threats, backdoors, and exploits
  • Published on arXiv (2507.10898)
RESULT
  • Published at IEEE IRI 2025
  • Demonstrated high-precision malware detection with explainable reasoning
  • System provides actionable fix recommendations alongside threat identification
  • Evolved into SecureFixAgent (the next iteration with agentic repair capabilities)
