AERIAL ROOF MATERIAL DETECTION

From the sky. For the ground.

SITUATION
  • Urban planners and insurance companies need accurate roof material data for disaster preparedness and risk assessment
  • Manual inspection is expensive and doesn't scale across large urban areas
  • Existing aerial imagery classifiers lacked the precision needed for material-level classification
TASK
  • Build a deep learning system to classify roof materials from aerial/satellite imagery
  • Improve detection accuracy significantly over existing baselines
  • Target real-world utility for urban analytics and disaster response
ACTION
  • Trained YOLOv8 on custom annotated aerial imagery dataset of roof materials
  • Implemented data augmentation, transfer learning, and hyperparameter optimization
  • Benchmarked against YOLOv5 baseline with comprehensive evaluation metrics
  • Presented at IEEE ICCDS 2024
RESULT
  • Won Best Paper Award at IEEE ICCDS 2024
  • Improved mAP by 32.5% over YOLOv5 baseline
  • Demonstrated practical applicability for insurance underwriting and municipal infrastructure assessment
  • First published paper, set the standard for problem selection going forward

TECH STACK

Python YOLOv8 PyTorch Computer Vision Aerial Imagery
