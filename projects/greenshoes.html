GREENSHOES

Luxury meets sustainability. Full stack.

SITUATION
  • Sustainable fashion brands need e-commerce platforms that communicate their environmental values alongside product details
  • Off-the-shelf platforms don't offer the customization needed for environmental impact metrics per product
  • A full-stack solution was needed: customer storefront, backend API, and admin dashboard
TASK
  • Build a complete e-commerce ecosystem from scratch for a luxury sustainable footwear brand
  • Implement product catalog with environmental impact metrics, shopping cart, checkout, user auth, and wishlists
  • Create a separate admin panel for product CRUD, inventory management, and order tracking
ACTION
  • Frontend: React 18 + Vite + Tailwind with filtering (category, size, color, price), product detail pages, and responsive design
  • Backend: Node.js + Express + Prisma ORM + PostgreSQL with auth, product, order, and inventory endpoints
  • Admin: Separate React app with role-based auth (ADMIN check), product CRUD, inventory by size/color variant, order tracking
  • All three repos independently deployable
RESULT
  • Complete, functional three-tier e-commerce platform
  • Customer app with environmental impact metrics per product
  • Admin dashboard with full inventory and order management
  • Clean separation of concerns across frontend, backend, and admin

TECH STACK

React Vite Tailwind CSS Node.js Express Prisma PostgreSQL Axios
