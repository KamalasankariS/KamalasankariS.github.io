FASS
Role: Primary Author
Co-Authors: Jugal Gajjar
Venue: CVPR 2026
About the Paper

Introduces the Feature Attribution Stability Suite, a comprehensive benchmark for measuring how stable post-hoc attribution methods are under real-world perturbations. The suite enforces prediction-invariance filtering to isolate explanation instability from model instability.

Problem Statement

Existing XAI evaluation lacks standardized stability metrics. Explanations can change drastically under minor input perturbations even when the model prediction stays the same, undermining trust in deployed AI systems.

pg. 23
Datasets Used

ImageNet validation set with geometric (rotation, flip, crop), photometric (brightness, contrast, blur), and compression (JPEG quality) perturbation pipelines.

Results

Grad-CAM shows highest stability across all perturbation types. GradientSHAP and LIME exhibit significant attribution drift. IG provides moderate stability. Results reveal method-level gaps previously unreported.

"stability matters more than accuracy when humans are in the loop"

pg. 24
HyperComplEx
Role: Co-Author
Co-Authors: Jugal Gajjar, Kaustik Ranaware, Vaibhav C. Gandhi
Venue: IEEE 2025
About the Paper

Proposes a multi-space knowledge graph embedding model that simultaneously operates in hyperbolic, complex, and Euclidean geometries to capture diverse relational patterns in a single unified architecture.

Problem Statement

Single-geometry KG embeddings fail to capture the full spectrum of relation types. Hierarchical relations need hyperbolic space, rotational patterns need complex space, and translational patterns need Euclidean space.

pg. 31
Datasets Used

WN18RR, FB15k-237, and YAGO3-10 knowledge graph benchmarks for link prediction evaluation.

Results

Outperforms RotatE, ComplEx, and MuRP on MRR and Hits@10 metrics. The multi-space fusion captures patterns that no single geometry can represent alone.

"one space to rule them all? no — three."

pg. 32
VulnGraph
Role: Co-Author
Co-Authors: Jugal Gajjar, Kaustik Ranaware
Venue: Complex Networks 2025
About the Paper

Develops a multimodal vulnerability detection model that fuses code structure (AST/CFG graphs) with semantic understanding (LLM embeddings) through a two-way gating mechanism, enabling interpretable and accurate detection.

Problem Statement

Existing detectors rely on either structural or semantic features alone. Structural approaches miss semantic context; LLM-based approaches miss control flow patterns. Neither produces interpretable explanations.

pg. 39
Datasets Used

Devign, Reveal, and BigVul multi-language vulnerability datasets spanning C, C++, Java, and Python codebases.

Results

93.57% detection accuracy. Saliency subgraphs identify the exact vulnerable code regions. Outperforms Devign, Reveal, and LineVul baselines across all datasets.

"the graph sees what the LLM misses"

pg. 40
SecureFixAgent
Role: Co-Author
Co-Authors: Jugal Gajjar, Relsy Puthal, Kaustik Ranaware
Venue: ICMLA 2025
About the Paper

Presents an autonomous agent that combines LLM code reasoning with multi-armed bandit optimization to detect, patch, and validate software vulnerabilities with explainable rationales for each repair decision.

Problem Statement

Manual vulnerability repair is slow and error-prone. LLMs alone generate plausible but often incorrect patches. There's no feedback loop to learn from failed repair attempts.

pg. 47
Datasets Used

CVEfixes, Big-Vul, and PatchDB datasets covering real-world CVEs across C/C++ and Java codebases.

Results

+13.5% patch success rate over baseline LLM repair. -10.8% false positive reduction. Explainable patch rationales validated by security experts.

"let the bandit learn which patches actually work"

pg. 48
MalCodeAI
Role: Co-Author
Co-Authors: Jugal Gajjar, Noha El Kachach
Venue: IEEE IRI 2025
About the Paper

Designs a two-phase LLM pipeline: Phase 1 performs deep semantic code reasoning to understand intent, Phase 2 simulates exploit scenarios to confirm malicious behavior. Automatically generates patches for detected threats.

Problem Statement

Traditional malware detection relies on signatures and heuristics that fail against obfuscated or zero-day code. Semantic understanding of code intent is needed for generalizable detection.

pg. 55
Datasets Used

Custom-curated dataset of malicious code samples across Python, JavaScript, and C, paired with benign counterparts for contrastive evaluation.

Results

Detects obfuscated malicious patterns that signature-based tools miss. Generates functional patches validated against exploit test suites. Language-agnostic across all tested codebases.

"understand intent first, then simulate the attack"

YOLO: Roof Material Detection
Role: Primary Author
Co-Authors: R. Avudaiamal, Neha Shirlin Christopher
Venue: IEEE ICCDS 2024 • Best Paper Award
About the Paper

Applies YOLOv8 object detection to aerial drone imagery for automated classification of rooftop materials including metal, asphalt, tile, and concrete. Designed for scalable urban infrastructure assessment and disaster resilience planning.

Problem Statement

Manual roof surveys are expensive, slow, and dangerous. Satellite imagery lacks the resolution for material-level classification. Drone-based automated detection can scale to city-wide assessment.

pg. 63
Datasets Used

Custom aerial drone imagery dataset with annotated rooftop regions across residential and commercial areas, covering 4 material categories with 5,000+ labeled instances.

Results

High mAP across all roof material classes. Real-time inference on drone hardware. Recognized with the Best Paper Award at IEEE ICCDS 2024 for practical impact on urban planning and disaster response.

"looking down to look ahead — literally"

Stack: Python, PyTorch, Grad-CAM, IG, SHAP, LIME
